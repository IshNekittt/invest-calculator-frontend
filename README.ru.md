# 📊 UI инвестиционного калькулятора (Монте Карло & GBM)

Клиентская часть SPA (Single Page Application) для визуализации стохастических финансовых расчетов на основе Геометрического броуновского движения.

### 🛠 Стек и библиотеки

- **React 19 & Vite** — Ядро и сборка.
- **Recharts** — Рендеринг графиков (коридор вероятностей).
- **Axios** — Взаимодействие с REST API.
- **React Router v6** — Маршрутизация.
- **UUID** — Теневая авторизация (сохраняет уникальный `deviceId` в LocalStorage).

### 📂 Структура проекта

- `src/pages/Home` — Страница с описанием математической модели.
- `src/pages/Calculator` — Основной интерфейс: форма ввода, график `Recharts` и блок истории.
- Форматирование чисел и графиков адаптировано под локаль (`uk-UA`).

```text
src/
 ├── components/
 ├── pages/
 │   ├── Home/
 │   └── Calculator/
 ├── App.jsx
 └── index.css
```

### 🚀 Cкрипты

- `npm install` — Установка зависимостей.
- `npm run dev` — Локальная разработка.
- `npm run build` — Сборка production-версии.
