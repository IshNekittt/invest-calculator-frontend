# 📊 UI інвестиціонного калькулятору (Монте Карло & GBM)

Клієнтська частина SPA (Single Page Application) для візуалізації стохастичних фінансових розрахунків на основі Геометричного броунівського руху.

### 🛠 Стек та бібліотеки

- **React 19 & Vite** — Ядро та збірка.
- **Recharts** — Рендеринг графіків (коридор ймовірностей).
- **Axios** — Взаємодія з REST API.
- **React Router v6** — Маршрутизація (навігація між описом моделі та калькулятором).
- **UUID** — Тіньова авторизація (зберігає унікальний `deviceId` у LocalStorage).

### 📂 Структура проєкту

- `src/pages/Home` — Сторінка з описом математичної моделі (GBM, Лемма Іто).
- `src/pages/Calculator` — Основний інтерфейс: форма введення, графік `Recharts` і блок історії.
- Форматування чисел і графіків адаптовано під локаль (`uk-UA`).

```text
src/
 ├── components/
 ├── pages/
 │   ├── Home/
 │   └── Calculator/
 ├── App.jsx
 └── index.css
```

### 🚀 Cкрипти

- `npm install` — Встановлення залежностей.
- `npm run dev` — Запуск локального сервера розробки.
- `npm run build` — Оптимізована збірка проєкту для продакшену.
